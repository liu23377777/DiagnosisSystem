第一部分：舌质（舌体）诊断标准
═══════════════════════════════════════════

一、舌色分类（9种）

1. 淡白舌
   【特征】舌色较正常浅淡，甚至白色，舌质干燥或湿润
   【主病】气血两虚、阳虚寒盛
   【常见症状】面色萎黄、疲乏无力、畏寒肢冷、气短懒言
   【治疗原则】温补脾肾、益气养血
   【推荐方剂】四君子汤、八珍汤、归脾汤
   【饮食建议】红枣、桂圆、山药、黄芪炖鸡
   【禁忌】生冷寒凉、过食苦寒

2. 淡红舌（正常舌象）
   【特征】舌色淡红鲜明、滋润、大小适中、灵活自如
   【主病】正常舌象，气血调和
   【说明】脏腑功能正常，营卫调和
   【保健建议】维持健康生活方式，均衡饮食

3. 红舌
   【特征】舌色鲜红
   【主病】热证（实热或虚热）
   【分类】
   - 舌尖红：心火上炎（伴失眠、口舌生疮）
   - 舌边红：肝胆有热（伴口苦、胁痛）
   - 全舌红：肺胃热盛（伴咳嗽、口渴）
   【治疗原则】清热降火
   【推荐方剂】导赤散、龙胆泻肝汤、清胃散
   【饮食建议】绿豆、苦瓜、西瓜、莲子心
   【禁忌】辛辣燥热、煎炸油腻

4. 绛舌
   【特征】舌色深红如猪肝，干燥少津
   【主病】热入营血、阴虚火旺
   【严重程度】较红舌更重
   【治疗原则】滋阴清热、凉血养阴
   【推荐方剂】清营汤、知柏地黄丸
   【注意】需及时就医，可能提示重症

5. 紫舌
   【特征】舌色紫暗或青紫
   【分类】
   - 紫暗舌：血瘀（舌质暗紫，无光泽）
   - 青紫舌：寒凝（舌质青紫，湿润）
   【主病】血瘀、寒凝、气滞
   【治疗原则】活血化瘀、温阳散寒
   【推荐方剂】血府逐瘀汤、温经汤
   【饮食建议】山楂、玫瑰花茶、当归生姜羊肉汤
   【禁忌】久坐不动、寒凉冰冷

6. 瘀斑舌
   【特征】舌边或舌面有青紫色瘀斑瘀点
   【主病】肝经血瘀、瘀血内停
   【常见症状】胸胁刺痛、月经血块、面色晦暗
   【治疗原则】活血化瘀
   【推荐方剂】桃红四物汤、膈下逐瘀汤

二、舌形特征（7种）

1. 胖大舌
   【特征】舌体肥大，充满口腔，伸舌困难
   【常伴】边缘有齿痕
   【主病】脾虚湿盛、痰湿内阻、水湿停留
   【常见症状】身重困倦、胸闷腹胀、大便溏薄
   【治疗原则】健脾化湿、利水消肿
   【推荐方剂】参苓白术散、五苓散
   【饮食建议】薏米、红豆、山药、茯苓
   【禁忌】甜腻、油腻、生冷

2. 齿痕舌
   【特征】舌边有牙齿压迫痕迹，呈锯齿状
   【主病】脾虚气虚、水湿停留
   【病机】脾虚则舌体胖大，齿压成痕
   【治疗原则】补气健脾、化湿利水
   【推荐方剂】四君子汤、补中益气汤
   【饮食建议】黄芪、党参、白术、大枣
   【禁忌】过劳、熬夜、思虑过度

3. 瘦薄舌
   【特征】舌体瘦小单薄，舌面少苔
   【主病】阴虚、血虚、气血不足
   【常见症状】消瘦、盗汗、潮热、失眠
   【治疗原则】滋阴养血、补益气血
   【推荐方剂】六味地黄丸、当归补血汤
   【饮食建议】百合、银耳、鸭肉、桑葚
   【禁忌】辛辣、煎炸、温燥

4. 裂纹舌
   【特征】舌面有各种形状的裂纹或裂沟
   【分类】
   - 阴虚型：舌红少苔有裂纹
   - 血虚型：舌淡有裂纹
   - 脾虚型：舌淡胖有裂纹
   【治疗原则】根据虚损类型补虚
   【注意】先天性裂纹舌不属病态

5. 点刺舌
   【特征】舌尖或舌面有红点突起，如草莓状
   【主病】热毒炽盛、气血壅滞
   【严重程度】点刺越多越红，热毒越重
   【治疗原则】清热解毒
   【推荐方剂】普济消毒饮、黄连解毒汤
   【注意】需及时就医

6. 歪斜舌
   【特征】伸舌时舌体偏向一侧
   【主病】中风先兆、风痰阻络
   【紧急程度】★★★★★
   【处理】立即就医，可能是中风

7. 强硬舌
   【特征】舌体僵硬不灵活
   【主病】热入心包、痰浊阻络
   【紧急程度】★★★★
   【处理】及时就医

第二部分：舌苔诊断标准
═══════════════════════════════════════════

一、苔色分类（6种）

1. 白苔
   【分类】
   - 薄白苔（正常）：能透见舌质，如一层白雾
   - 白厚苔：不能透见舌质，如积粉
   - 白腻苔：白厚且粘腻，刮之不去
   【主病】
   - 薄白苔：正常或表证初起
   - 白厚腻苔：寒湿、痰饮、食积
   【治疗】根据厚薄和腻滑程度选方

2. 黄苔
   【特征】舌苔呈黄色，颜色深浅不一
   【分类】
   - 薄黄苔：表邪入里、热证初起
   - 黄厚苔：里热炽盛、痰热互结
   - 焦黄苔：热盛伤阴、胃肠积热
   【主病】热证（颜色越深，热越重）
   【治疗原则】清热泻火
   【推荐方剂】黄连解毒汤、大承气汤

3. 灰苔
   【特征】舌苔呈灰色
   【主病】里证（寒湿或郁热）
   【鉴别】需结合舌质红淡判断寒热
   - 舌淡苔灰：寒湿内盛
   - 舌红苔灰：热郁化燥

4. 黑苔
   【特征】舌苔呈黑色或焦黑色
   【主病】热极或寒盛
   【严重程度】★★★★★
   【鉴别】
   - 焦黑干裂：热极伤阴
   - 黑润滑：寒盛
   【处理】需紧急就医

5. 剥苔（花剥苔）
   【特征】舌苔部分剥落，露出光滑舌质
   【主病】胃气胃阴不足、气血两虚
   【常见】慢性胃炎、消化不良
   【治疗原则】益胃生津、养阴生津
   【推荐方剂】益胃汤、麦门冬汤

6. 镜面舌（无苔）
   【特征】舌面光滑如镜，完全无苔
   【主病】胃气胃阴枯涸
   【严重程度】★★★★
   【治疗原则】滋阴养胃
   【推荐方剂】沙参麦冬汤

二、苔质分类（4类）

1. 厚薄
   【判断标准】
   - 薄苔：能透见舌质（正常或病邪在表）
   - 厚苔：不能透见舌质（邪气深重）
   【临床意义】
   - 苔由薄变厚：病邪加重、邪气入里
   - 苔由厚变薄：正气恢复、病情好转

2. 润燥
   【润苔】
   - 特征：苔面湿润，有适度津液
   - 意义：津液未伤
   【燥苔】
   - 特征：苔面干燥，缺少津液
   - 主病：津液损伤、阴虚
   【治疗】生津养阴

3. 腻滑
   【腻苔】
   - 特征：苔质粘腻，颗粒细密，刮之不去
   - 主病：湿浊、痰饮、食积
   - 治疗：化湿、祛痰、消食
   【滑苔】
   - 特征：苔面水滑，摸之湿润多津
   - 主病：水湿内停、痰饮
   - 治疗：温阳化饮、利水渗湿

4. 腐腻
   【腐苔】
   - 特征：苔质疏松如豆腐渣，易刮去
   - 主病：食积化腐、痰浊化热
   - 治疗：消食导滞
   【腻苔】
   - 特征：苔质致密，不易刮去
   - 主病：痰湿、湿浊
   - 治疗：化湿祛痰

第三部分：常见证型速查表
═══════════════════════════════════════════

证型1：脾胃气虚
【舌象】淡红舌，齿痕明显，薄白苔
【主症】食少纳呆、腹胀便溏、神疲乏力、少气懒言
【兼症】面色萎黄、自汗、脱肛
【治法】健脾益气
【方剂】四君子汤、补中益气汤、参苓白术散
【饮食】山药、大枣、扁豆、黄芪炖鸡
【禁忌】生冷、油腻、过饱

证型2：脾虚湿盛
【舌象】淡胖舌，边有齿痕，白厚腻苔
【主症】肢体困重、胸闷纳呆、大便溏薄、腹胀
【兼症】头重如裹、口淡不渴、小便不利
【治法】健脾化湿
【方剂】参苓白术散、香砂六君子汤、平胃散
【饮食】薏米、红豆、茯苓、陈皮
【禁忌】甜腻、生冷、油腻

证型3：阴虚火旺
【舌象】红绛瘦舌，少苔或剥苔，舌面干燥有裂纹
【主症】五心烦热、盗汗、失眠多梦、口干咽燥
【兼症】腰膝酸软、遗精、经少
【治法】滋阴降火
【方剂】知柏地黄丸、麦味地黄丸、大补阴丸
【饮食】百合、银耳、鸭肉、桑葚、梨
【禁忌】辛辣、煎炸、温燥、熬夜

证型4：湿热内蕴
【舌象】红舌，黄厚腻苔
【主症】口苦口黏、大便黏滞不爽、小便黄赤、身热不扬
【兼症】胸闷腹胀、恶心、肢体困重
【治法】清热化湿
【方剂】龙胆泻肝汤、茵陈蒿汤、三仁汤
【饮食】绿豆、冬瓜、苦瓜、芹菜
【禁忌】辛辣、油腻、酒类、海鲜

证型5：寒湿困脾
【舌象】淡胖舌，白滑苔或白腻苔
【主症】脘腹冷痛、大便溏薄、肢体困重、畏寒喜暖
【兼症】纳呆、口淡不渴、小便清长
【治法】温中化湿
【方剂】胃苓汤、藿香正气散、理中丸
【饮食】生姜、干姜、茴香、肉桂
【禁忌】生冷、寒凉、过食瓜果

证型6：气滞血瘀
【舌象】紫暗舌或有瘀斑瘀点，薄苔
【主症】胸胁刺痛、痛有定处、舌下络脉青紫
【兼症】月经不调、经色暗有血块、面色晦暗
【治法】活血化瘀、行气止痛
【方剂】血府逐瘀汤、桃红四物汤、膈下逐瘀汤
【饮食】山楂、玫瑰花、三七、丹参
【禁忌】久坐不动、情志不畅

证型7：痰湿阻滞
【舌象】淡红胖大舌，边有齿痕，白厚腻苔
【主症】痰多胸闷、头晕目眩、恶心呕吐、身重困倦
【兼症】咳嗽痰多、纳呆、便溏
【治法】化痰祛湿
【方剂】二陈汤、温胆汤、苓桂术甘汤
【饮食】陈皮、半夏、茯苓、薏米
【禁忌】肥甘厚味、甜腻、油炸

证型8：气血两虚
【舌象】淡白舌，舌体瘦薄，薄白苔
【主症】面色萎黄、神疲乏力、心悸失眠、头晕目眩
【兼症】少气懒言、自汗、月经量少
【治法】补益气血
【方剂】八珍汤、归脾汤、人参养荣汤
【饮食】红枣、桂圆、阿胶、当归炖鸡
【禁忌】过劳、熬夜、暴饮暴食

证型9：肝火上炎
【舌象】舌边尖红，舌苔黄
【主症】头痛、目赤肿痛、口苦、急躁易怒
【兼症】耳鸣如潮、胁痛、失眠多梦
【治法】清肝泻火
【方剂】龙胆泻肝汤、丹栀逍遥散
【饮食】菊花、决明子、芹菜、苦瓜
【禁忌】辛辣、烟酒、情绪激动

第四部分：舌诊辨证要点
═══════════════════════════════════════════

【舌诊望诊顺序】
1. 先看舌质（颜色、形态）
2. 再看舌苔（颜色、厚薄、润燥、腻滑）
3. 最后看舌下络脉

【舌质主脏腑气血】
- 舌色：反映气血盛衰、寒热性质
- 舌形：反映脏腑虚实
- 舌态：反映病情进退

【舌苔主邪气深浅】
- 苔色：反映病邪性质（寒热）
- 苔质：反映邪气盛衰、津液存亡

【重要鉴别点】
1. 红舌与绛舌：红为热盛，绛为热深
2. 腻苔与滑苔：腻为痰湿，滑为水饮
3. 瘦舌与裂舌：瘦为阴虚血虚，裂为阴液亏损
4. 胖舌与齿痕：都提示脾虚，齿痕更明显

【诊断思路】
第一步：确定舌质颜色（淡白/淡红/红/绛/紫）
第二步：观察舌体形态（胖/瘦/齿痕/裂纹）
第三步：分析舌苔颜色（白/黄/灰/黑/剥）
第四步：判断舌苔性质（厚薄/润燥/腻滑）
第五步：综合辨证，确定证型

【注意事项】
1. 避免在刚进食、喝咖啡后观察
2. 自然光线下观察最准确
3. 伸舌过度会影响舌色
4. 药物（抗生素）可能影响苔色
5. 染苔（食物染色）需排除

第五部分：治疗调护建议
═══════════════════════════════════════════

【饮食调护总原则】
1. 脾虚者：多食山药、大枣、薏米、扁豆
2. 阴虚者：多食百合、银耳、梨、鸭肉
3. 阳虚者：多食羊肉、生姜、韭菜、核桃
4. 湿盛者：多食薏米、红豆、陈皮、茯苓
5. 血瘀者：多食山楂、红花、三七、黑木耳

【生活调护】
1. 保持规律作息，避免熬夜
2. 适度运动，促进气血运行
3. 保持心情舒畅，避免情志过激
4. 饮食有节，不可过饱过饥
5. 根据季节调整饮食起居

【中医特色疗法】
1. 针灸：调理脏腑功能
2. 推拿：疏通经络
3. 拔罐：祛除湿邪
4. 艾灸：温阳散寒
5. 刮痧：活血化瘀

【就医提示】
出现以下舌象需及时就医：
1. 绛舌、黑苔：提示病情严重
2. 歪斜舌：中风征兆
3. 强硬舌：热入心包
4. 点刺舌：热毒炽盛
5. 镜面舌：胃阴枯涸
